# nisrin-site

Personal portfolio site for Nisrin Shoukat Attarwala: RegTech and compliance automation projects, writing, and a downloadable CV.

## Stack

React, TypeScript, Vite, Tailwind CSS. Fully static, deployed on Vercel.

## Develop

```
pnpm install
pnpm dev
```

## Build

```
pnpm build
```

## Regenerate the LinkedIn QR code

```
pnpm gen:qr
```
